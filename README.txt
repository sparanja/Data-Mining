Libraries: sckit, pandas, numpy, datetime, matplotlib.
Function: Does data preprocessing, feature extraction and
          dimesionality reduction using PCA.

Steps to execute:
* Copy the CGM.py to the DataFolder
* Run python CGM.py on command line
* If there are errors relating to uninstalled libraries, perform(For example):
	pip install pandas
  Or any other missing libraries
* The graph plots are stored in plots/ folder
* Make sure to delete the plots on every run for different patients(optional)
* Output will be saved in Patient(X)Merged.csv
